# Sustainable Ways of Growing Food — Group Project

Simple static website for a group assignment about sustainable food-growing methods.

How to use
- Open `index.html` in a browser (double-click or serve with a simple server).

To serve locally (Python):

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

Files added
- `index.html` — main site
- `styles.css` — styles
- `script.js` — small interactivity (nav + smooth scroll)
